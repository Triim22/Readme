**Descrição:**
Este projeto tem como objetivo:

**Objetivos:**
* Facilitar o Agendamento : Tornando-o mais intuitivo e fácil, permitindo que os usuários encontre horários disponíveis e confirmando seus atendimentos rápido e com simplicidade.
* Aumentar a Eficacia : Otimizando a gestão dos horários e recursos do NAF, reduzindo o tempo de espera dos usuários.
* Modernizar o NAF : Posicionar o NAF como uma instituição inovadora e eficiente, utilizando tecnologia para melhorar a qualidade dos servisos.

**Benefícios:**
* Confirmação por E-mail: Enviar um e-mail de confirmação ao usuário com os detalhes do agendamento (data, hora, serviço solicitado).
* Informções mais claras.
* Agendamento Online: Permitir que os usuários selecionem um horário disponível e confirmem o agendamento.

## MVP
Cadastro de Usuários: Permitir que os usuários se cadastrem no sistema, fornecendo informações básicas como nome, e-mail e senha
Consulta de Horários Disponíveis: Apresentar um calendário com os horários disponíveis para agendamento, considerando a capacidade de atendimento do NAF.
Painel Administrativo Simples: Permitir que os administradores do NAF visualizem a agenda, confirmem agendamentos e gerenciem os dados dos usuários.
